Sum of square numbers

Given a non-negative integer c, decide whether there're two integers a and b such that:
a^2 + b^2 = c.

Example 1:
Input: c = 5
Output: true
Explanation: 1 * 1 + 2 * 2 = 5

Example 2:
Input: c = 3
Output: false
Explanation: 1 * 1 + 1 * 1 = 2, 2 * 2 + 0 * 0 = 4

Constraints:
0 < = c < = 2^31 - 1
